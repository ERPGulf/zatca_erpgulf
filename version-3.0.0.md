Version 3.0 release notes

Highlights:

    Optimized XML Handling: In this release, the system now generates and saves only a single XML file per invoice,
    Eliminating the creation of temporary files.

Hard Disk Efficiency: 

    By removing temporary XML storage, disk space usage is minimized, ensuring better performance and reduced storage overhead.

    Improved Performance: Faster XML generation and reduced I/O operations on the server.

Impact:

    Users will notice a cleaner file structure with only the essential XML file saved.

    Improved server storage management, especially for high-volume invoice generation.

Upgrade Notes:

    No additional configuration is required. Simply update to version 3.0 to benefit from the new optimized XML handling.
